A. Macam - Macam Tag dan Fungsinya

Tag yang berada pada head:

1. Tag Title
Untuk judul dokumen yang ditampilkan dibilah judul browser atau di tab halaman, harus berupa teks saja.
2. Tag Style
Untuk menentukan informasi gaya untuk satu halaman HTML.
3. Tag Meta
Untuk menentukan kumpulan karakter, deskripsi halaman, kata kunci, penulis dokumen, dan pengaturan area pandang. Metadata tidak akan ditampilkan pada halaman, tetapi digunakan oleh browser, oleh mesin pencari, dan layanan web lainnya.
4. Tag Link
Hubungan antara dolumen saat ini dengan sumber daya eksternal. Sering untuk menghubungkan ke style sheet eksternal.
5. Tag Script
Untuk mendefinisikan JavaScript sisi klien.
6. Tag Base
Menentukan URL dasar dan/atau target untuk semua URL relatif di suatu halaman. Dan hanya ada satu Tag Base dalam sebuah dokumen.

Tag yang berada pada body:

1. Teks
Tag h1 (paling besar) - tag h6 (paling kecil), tag p (paragraf), dll...
2. Pendukung Teks
Tag br (menambahkan baris baru), tag hr (menambahkan garis horizontal), tag em, tag stong, dll...
3. Gambar
Tag img, untuk meletakkan image pada dolumen HTML.
4. Hyperlink
Tag a, untuk meletakkan link pada HTML.
5. List (bullets dan Numbering)
Tag uk (untuk list tidak terurut), tag ol, tag li (tag yang berada di dalam ul dan ol), tag dl, tag dt, dan tag dd.
6. Tabel
Tag table, tag thead, tag tbody, dll...
7. Form
Tag form, tag input, tag select, tag button, dll...
8. Script
Tag script (JavaScript)
9. Object
Tag object
10. Groping
Tag div dan tag span
